Sorting Hat (Python)

Created with Python and packaged as a standalone .exe, Sorting Hat is a simple but powerful tool for extracting and sorting numbers from any input data. With a simple and user-friendly interface, the GUI allows users to easily extract numbers from data then sort them and also display them on a bar graph, line graph, scatter plot, histogram, or pie chart giving flexibility to choose the best way to visualize the data.
